Key Commands

npm run format Check formatting (Oxfmt)
npm run format:fix Auto-format all files
npm run lint Lint TS/JS files (Oxlint)
npm run lint:fix Auto-fix lint issues
npm run lint:html Lint Angular HTML templates
npm run lint:styles Lint CSS & SCSS files
